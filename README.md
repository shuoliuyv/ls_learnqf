## draft

使用一个包含约 500 只印度股票的 1 分钟级别 OHLCV 数据（2015–2026），主要用来：

* 练习基础的数据处理
* 做一些比较简单的因子测试
* 实现金融中常见的指标（如波动率、MACD、RSI）
* 熟悉量化研究的基本流程（目前更多还是在“找感觉”的阶段）

## utils

一些常用的工具函数，主要包括：

* 股票抓取，数据清洗， 因子处理与简单检验（如去极值、标准化、中性化等），基础的回测相关函数以及一些零散但常用的小工具。

## 各个文件夹

目前主要在复现和尝试的一些因子与策略，文件夹内附研报 pdf。选取这些研报是因为在当前比较简陋的数据与框架下，基本覆盖了横截面因子、择时/时序因子、多因子以及多因子组合优化等核心要素。后续也会陆续补充其他策略复现以及这学期使用 C++ 实现的高频交易系统课程项目相关内容。

### Amplitude
股票存在低波动异象，基于价格振幅构建因子，并进一步按价格维度切割（高价 / 低价），刻画振幅的结构差异，在此基础上构造改进因子以提升选股稳定性。

### RSRS
通过对最高价与最低价进行回归来刻画支撑/阻力强度，从而构建择时信号；同时针对震荡市场中信号不稳定的问题，引入参数调整与指标钝化进行改进。

### 自适应风控指数增强
在多因子选股与指数增强框架下，引入动态风险控制机制，根据历史跟踪误差与市场波动对个股偏离度进行调整，使组合在不同市场环境下表现更稳定。

### 行业量价因子 / 行业轮动
在行业层面构建多类量价因子（如动量、波动、换手等），进行单因子筛选并合成复合因子，对行业进行排序与轮动配置，以捕捉行业之间的相对强弱。
